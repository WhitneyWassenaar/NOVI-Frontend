# NOVI Frontend
___
## Course overview
___
This course teaches how to design interactive systems that are centered around the user. I will design logical interfaces and create prototypes in Figma. The course also covers visual design using HTML and CSS.

I will learn how to use AJAX to communicate with external data sources and implement them in web pages using reusable user interface components. By the end of the course, I will be able to build a complete web application from start to finish.

- HTML & CSS
- Javascript 
- React
